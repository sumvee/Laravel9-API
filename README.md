## Installation
```
vendor/bin/sail up -d (assuming docker is available)
configure /etc/hosts file to point http://laravel-api.test/ to docker exposed IP
Import  API.postman_collection.json in root folder to use postman
Curl requests in curl_requests in root folder 

```
